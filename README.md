# Komiscope â€“ Offline Comic Reader (Android Studio)

Komiscope adalah aplikasi pembaca komik offline berbasis Android yang mendukung file **CBZ** dan **ZIP**.  
Aplikasi ini dikembangkan sebagai project Ujian Tengah Semester mata kuliah **Pemrograman Bergerak**.

##  Fitur Utama
- Membaca komik offline dari file CBZ dan ZIP
- Navigasi halaman menggunakan swipe
- Zoom gambar dengan pinch gesture
- Penyimpanan progres membaca
- Error handling untuk file tidak valid atau tanpa gambar

##  Teknologi yang Digunakan
- Kotlin
- Android SDK
- ViewPager2
- PhotoView
- Glide
- DataStore Preferences
- Coroutines
- Material Design 3

##  Cara Menjalankan
1. Jalankan aplikasi
2. Pilih file komik berformat CBZ atau ZIP [File Tester Di sini](https://drive.google.com/drive/folders/1LIc2YdyU1ypDHZC2U-VfCKCqRHATa36X?usp=drive_link)
3. Aplikasi akan menampilkan halaman komik secara berurutan
4. Gunakan swipe untuk berpindah halaman dan pinch untuk zoom

##  Studi Kasus
Aplikasi ini menerapkan konsep **Artificial Intelligence klasik** dalam bentuk *intelligent content handling* berbasis aturan (rule-based system), seperti:
- Pengenalan struktur file arsip
- Penyaringan otomatis konten gambar
- Pemulihan konteks membaca pengguna

##  Note
This application is designed for offline use and does not perform scraping or data retrieval from the internet.
